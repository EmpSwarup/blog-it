## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Then install necesscary packages and libraries

```bash
npm install
# or
yarn install
# or
pnpm install
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
